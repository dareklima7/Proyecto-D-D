# Proyecto-D-D

Este proyecto es una pequeña aplicación en Java donde se crean personajes de tipo Guerrero y Mago, todos heredando de una clase base llamada Personaje.
El objetivo principal es practicar herencia, clases abstractas, sobrescritura de métodos y el uso correcto de Git y GitHub con ramas.

Qué hace el programa?
Crea un Guerrero y un Mago.
Cada personaje tiene: nombre, vida, ataque, defensa, Ambos pueden atacar a otro personaje.
El programa muestra: los datos de cada personaje, el daño recibido, la vida restante
